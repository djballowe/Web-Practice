HTML Practice Reference Table

Most Important CSS Tags

Tag Name Description
background-image Background Image Used to specify a background image
float Float Used for positioning and formating content e.g. let an image float left ot the text in a container
display Display Sets whether an element is treated as a block or inline element and the layout used for its children.
margin Margin Sets the margin area on all four sides of an element.
padding Padding Sets the padding area on all four sides of an element at once
overflow Overflow Sets the desired behavior for an elements overflow. i.e. when an elements content is too big to fit in its block formatting context.

Styling Text

Tag Name Description
font-style Font Style Used to turn italic text on or off
font-weight Font Weight Sets how bold the text is
text-transform Text Transform Allows you to set your font to be transformed
